# geosoft_1_final_project
## Task: 
Develops a map-based,  interactive, digital city guide. The  city guide is intended to facilitate the addition and  search for places of interest in  Muenster (NRW, Germany).

## Technologie
* Leaflet
* turf
* jquery
* ajax
* Bootstrap
* node, express
* npm
* docker
* mongoDB
